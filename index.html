CoreX Framework

A modern, modular FiveM framework built in Lua, designed for flexibility, performance, and ease of use.

v1.0.2.0 Alpha

CoreX Framework is the next generation of FiveM development, combining the best features from ESX, OX, and QBCore while introducing modern architecture and enhanced performance. Built with developers in mind, CoreX provides a solid foundation for creating amazing roleplay experiences.

Modular Architecture

Dynamic module loading system with clean separation of concerns

Multi-Language Support

Built-in localization system supporting 5 languages

Discord Integration

Seamless Discord bot integration for permissions and chat

Multi-Character System

Support for multiple characters per player with perma-death

Built-in MySQL Support

Native MySQL integration with connection string configuration

Security First

Role-based permissions and comprehensive audit logging

Get CoreX running on your server in minutes with our streamlined setup process.

1

Download CoreX

Clone the repository or download the latest release

Terminal 
git clone https://github.com/Core-X-Dev/CoreX.git
2

Configure Database

Add your MySQL connection string to server.cfg

server.cfg 
set mysql_connection "mysql://username:password@localhost:3306/database_name?"
ensure CoreX
3

Start Server

Start your FiveM server and CoreX will automatically initialize

No External Dependencies: CoreX handles all MySQL operations internally
